# SEC Survivor Basketball 2026

A mobile-first companion application for the 2026 SEC Basketball Survivor season, allowing participants to browse the completed season through interactive standings, player histories, weekly results, and season progression.

Unlike the football application, the 2026 basketball site serves as a historical archive of a completed season. It demonstrates how the survivor platform can be adapted to a different sport while preserving the same user experience, visualization, and analytics.

Beginning with the 2027 season, this application will evolve into a full-featured survivor platform with player authentication, online pick submission, and automated scoring powered by Supabase.

**Live Demo:** https://pollackelliott.github.io/sec-survivor-basketball-2026

# Overview

SEC Survivor Basketball recreates the 2026 survivor season through an interactive, mobile-first interface. Visitors can explore weekly standings, player timelines, pick histories, game results, and elimination progression exactly as the season unfolded.

The application demonstrates how a common platform can be adapted from football to basketball while accounting for differences in scheduling, scoring, and survivor rules.

Although this version is intentionally read-only, it serves as the foundation for the fully interactive basketball survivor application planned for the 2027 season.

# Relationship to SEC Survivor Football

This project was developed by adapting the architecture and interface of the SEC Survivor Football platform to support a completed basketball season.

While this version is read-only, it validates the platform's flexibility and serves as a stepping stone toward the fully automated basketball application planned for 2027.
